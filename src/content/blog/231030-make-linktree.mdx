링크를 모아 주는 서비스로는 LinkTree가 가장 유명할 것이다. 국산 서비스로 리틀리도 등장했으나,
결제하지 않으면, 무조건 링크 하단에 광고가 붙게 된다.
결제해서 쓰는 대신에 직접 구현해봤다.

처음에는 Next.js와 Vecel을 활용해서 구현했었다. 하지만, 링크가 모여 있는 한 페이지만 필요하므로, Next.js까지 필요 없다는 생각이 들었고,
Vite로 구현 후, GitHub Pages로 배포했다.

구현한 App.tsx의 코드는 아래와 같다.

```tsx
//App.tsx

import UserInfo from '@components/UserInfo';
import SocialLinkList from '@components/SocialLinkList';
import LinkList from '@components/LinkList';

function App() {
  return (
    <div className="bg-bkg flex min-h-screen flex-col p-2 text-white">
      <div className="container m-1 mx-auto flex max-w-xs flex-grow flex-col items-center gap-4 rounded-lg px-4 py-2">
        <UserInfo />
        <SocialLinkList />
        <hr className="w-full bg-white" />
        <LinkList />
      </div>
    </div>
  );
}

export default App;
```

사용자 정보를 제공해주는 UserInfo 와, SNS 아이콘 모음인 SocialLinkList, 각 링크 버튼 모음인 LinkList 세 개의 컴포넌트로 나눠 구현했다.
