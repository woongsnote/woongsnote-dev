`React`는 Facebook에서 개발된 UI 라이브러리로, 대규모 웹 애플리케이션을 빌드하는 데 사용된다.
Virtual DOM을 기반으로하며, 이를 사용하여 DOM 조작을 최소화하여 빠른 성능을 제공한다.

컴포넌트 기반 아키텍처를 사용하여 재사용 가능한 UI 구성 요소를 작성하는 데 초점을 둔다.
각 컴포넌트는 자체 상태와 생명주기 메서드를 가지며, 다른 컴포넌트와 조합하여 화면을 구성한다.

React는 `JSX`라는 문법을 사용하여 `JavaScript 코드`와 함께 `HTML`과 유사한 구문을 작성할 수 있도록 한다.
JSX는 보다 직관적이고 가독성이 높은 코드를 작성할 수 있게 해주며, React에서의 개발 생산성을 향상시킨다.

`React Router`는 React 애플리케이션의 라우팅을 관리하고, `Redux`는 애플리케이션의 상태를 효과적으로 관리하는 데 사용된다.
또한, `React Native`는 React를 사용하여 모바일 애플리케이션을 개발할 수 있는 도구다.

React는 커뮤니티와 사용자 수가 계속해서 늘어나고 있으며, 많은 기업들이 React를 사용하여 웹 애플리케이션을 개발하고 있다.
이러한 이유로 React는 현재 가장 인기있는 JavaScript 라이브러리 중 하나다.
