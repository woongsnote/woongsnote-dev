## Hook이란?

hook은 React 16.8 버전부터 도입된 새로운 기능이다. hook을 사용하면 클래스 컴포넌트를 작성하지 않고도 상태(state)와 생명주기(lifecycle) 메서드를 사용할 수 있다.
hook은 함수형 컴포넌트에서 React의 기능을 '연결(hook into)'해주는 것이라고 생각할 수 있다.

## Hook의 장점

- hook을 사용하면 컴포넌트를 더 간결하고 재사용하기 쉽게 만들 수 있다.

  예를 들어, useState hook은 상태 관리를 쉽게 해주고, useEffect hook은 부수 효과(side effect)를 관리할 수 있게 해준다.

- 커스텀 hook을 만들어서 다른 컴포넌트에서도 공통적으로 사용할 수 있는 로직을 분리할 수 있다.

## Hook을 사용하는 방법

hook은 함수형 컴포넌트 안에서만 호출할 수 있다.
또한, hook은 항상 최상위에서만 호출해야 한다. 즉, 반복문, 조건문, 중첩된 함수 안에서는 hook을 호출하면 안된다. 
이는 React가 hook이 호출되는 순서를 기억하고 상태를 올바르게 관리하기 위함이다.

hook에 대해 더 알고 싶다면, [공식 문서](https://react.dev/reference/react)를 참고하기 바란다.

## 커스텀 훅

커스텀 훅은 React의 기본 훅을 조합하여 재사용 가능한 로직을 만들 수 있는 방법이다.
커스텀 훅은 특별한 규칙이나 문법이 있는 것이 아니라, use로 시작하는 함수를 정의하는 것입니다.
커스텀 훅은 다른 컴포넌트에서도 사용할 수 있으므로 코드의 중복을 줄이고 관리하기 쉽습니다.

예를 들어, useInput이라는 커스텀 훅을 만들어보겠습니다. useInput은 input 태그의 value와 onChange 이벤트를 관리하는 훅입니다.
useInput은 다음과 같이 작성할 수 있습니다.

```js
import { useState } from "react";

function useInput(initialValue) {
  const [value, setValue] = useState(initialValue);

  const onChange = (e) => {
    setValue(e.target.value);
  };

  return { value, onChange };
}
```

useInput은 초기값을 인자로 받아서 value와 setValue를 useState로 생성합니다.
그리고 onChange 함수를 정의하여 e.target.value로 value를 업데이트합니다.
마지막으로 value와 onChange를 객체로 반환합니다.

useInput을 사용하는 방법은 다음과 같습니다.

```js
import React from "react";
import useInput from "./useInput";

function App() {
  const name = useInput(""); // 초기값은 빈 문자열
  const email = useInput(""); // 초기값은 빈 문자열

  return (
    <div>
      <input type="text" {...name} /> {/* name.value와 name.onChange를 전달 */}
      <input type="email" {...email} /> {/* email.value와 email.onChange를 전달 */}
    </div>
  );
}

export default App;
```

App 컴포넌트에서 useInput을 두 번 호출하여 name과 email을 생성한다.
각각의 초기값은 빈 문자열이고, input 태그에 `{...name}` 과 `{...email}`을 전달하여 value와 onChange를 바인딩한다.
이렇게 하면 input 태그의 값이 변경될 때마다 커스텀 훅의 value가 업데이트된다.
커스텀 훅을 사용하면 React의 기본 훅을 활용하여 재사용 가능한 로직을 만들 수 있고, 커스텀 훅은 다양한 상황에 적용할 수 있다.

## 결론

hook을 잘 활용하면 더 효율적이고 유지보수하기 쉬운 React 애플리케이션을 만들 수 있으므로 기본 훅과 커스텀 훅을 사용할 줄 알아야 한다.
