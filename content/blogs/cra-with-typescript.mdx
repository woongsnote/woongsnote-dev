### 프로젝트 생성

타입스크립트로 새 프로젝트를 생성하려면, 아래 명령어로 생성할 수 있다.
패키지 매니저에 따라 두 가지로 나누어진다.

```bash
npx create-react-app my-app --template typescript
```

또는

```bash
yarn create react-app my-app --template typescript
```

### 기존 프로젝트에 타입스크립트 적용

타입스크립트가 설치되지 않은 상태이므로, 아래 명령어로 타입스크립트를 설치해야 한다.

```bash
npm install --save typescript @types/node @types/react @types/react-dom @types/jest
```

또는

```bash
yarn add typescript @types/node @types/react @types/react-dom @types/jest
```

설치가 끝나면, 타입스크립트를 사용할 준비는 끝난다.

기존 js 파일을 ts/tsx 파일로 변경하고 개발 서버를 다시 시작하면, 타입이 지정되어 있지 않기 때문에 타입 오류가 발생할 것이다.

모든 타입 오류를 해결하면, 타입스크립트로의 마이그레이션이 마무리된다.

추가적인 내용은 아래 공식 문서를 참고하길 바란다.
