## 타입스크립트는?

**타입스크립트(TypeScript)** 는 자바스크립트에 타입을 추가한 정적 타입 언어이다.
자바스크립트로 작성된 프로그램을 타입스크립트로 변환할 수 있고, 타입스크립트 컴파일러를 사용하여 타입 오류를 미리 검출할 수 있다.

## 타입스크립트의 특징

- 변수, 매개변수, 함수 등의 값에 대해 **명시적으로 타입을 지정**하여 코드 작성 시 **타입 관련 실수를 방지**하고, **코드의 가독성**과 **유지보수성**을 높일 수 있다.

- **인터페이스**, **제네릭** 등의 기능을 제공하여 **코드의 재사용성**을 높인다.

- **객체지향 프로그래밍**의 특징을 지원한다.

- **자바스크립트와 호환**되며, 자바스크립트 코드를 그대로 사용할 수 있다.

- 컴파일 시 자바스크립트 코드로 변환되어, **모든 브라우저와 플랫폼에서 실행 가능**하다.

- Angular, React, Vue 등의 프론트엔드 프레임워크와 함께 많이 사용된다. 

- **마이크로소프트**에서 개발하고 관리하고 있다.
