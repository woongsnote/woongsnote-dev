## 소개
`Redux-toolkit`은 Redux의 공식적인 유틸리티 세트로, 보일러플레이트 코드를 줄이고 코드를 간결하고 읽기 쉽게 만들어주는 기능들을 제공한다.

### Redux toolkit의 주요 기능

1. 간결한 리덕스 코드 작성: 리덕스 툴킷은 `createAction`, `createReducer`, `createSlice`와 같은 메서드를 제공하여, 보일러플레이트 코드 작성을 줄여주고 간단하게 리덕스 액션과 리듀서를 작성할 수 있다.

2. 불변성 업데이트 간단화: 리덕스 툴킷은 immer 라이브러리를 사용하여 불변성 업데이트를 보다 쉽게 작성할 수 있도록 한다.

3. 비동기 작업 처리: 리덕스 툴킷은 Redux Thunk와 Redux Saga와 같은 비동기 미들웨어를 지원하며, 비동기 작업 처리를 간단하게 만들어준다.

4. 성능 최적화: 리덕스 툴킷은 자동으로 불변성을 유지하고, 불필요한 렌더링을 방지하기 위해 memoization을 사용하여 성능을 최적화한다.

### 결론

리덕스 툴킷은 리덕스의 개발자 경험을 개선하고 코드의 가독성과 유지 보수성을 향상시키기 위해 설계되었다. 
리덕스를 처음 사용하는 경우에도 빠르게 익힐 수 있으며, 대규모 애플리케이션에서도 유용하게 사용될 수 있다.
