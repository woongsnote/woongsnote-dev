title: React 상태관리 Tool의 비교
description: React에서 많이 사용되는 상태관리 Tool
date: 2023-02-20
---

리액트 상태 관리 툴은 여러 가지가 있지만 가장 대표적인 것으로 Redux, Recoil 등이 있다.

**Redux**는 가장 많이 사용되는 상태 관리 라이브러리 중 하나로, Flux 아키텍처를 기반으로 한다.
Redux는 단방향 데이터 흐름을 사용하며, 액션(Action) -> 리듀서(Reducer) -> 스토어(Store)로 데이터가 흐른다.

**Recoil**은 Facebook에서 개발한 새로운 상태 관리 라이브러리다.
Recoil은 이전 상태 관리 라이브러리와는 달리 컴포넌트 내에서 상태를 직접 정의하고 관리한다.
이를 통해 컴포넌트 간 상태 공유를 쉽게 할 수 있습니다.

이러한 상태 관리 라이브러리들은 각자 장단점이 있으므로, 프로젝트의 요구사항에 따라 적절한 라이브러리를 선택해야 한다.
